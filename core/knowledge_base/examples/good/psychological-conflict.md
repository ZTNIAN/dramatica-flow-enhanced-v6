# 内心冲突示例

## 类型1：道德冲突

```
他可以开门。

门外的姑娘在雨里，看起来快冻僵了。这种天气，这种时辰，让她走二十里夜路，跟送死没什么区别。

但他也可以不开。

这大半夜的，一个来历不明的女人，谁知道是什么来路？去年冬天，隔壁村的老王头就是半夜开了门，再也没人见过他。

老周的手放在门把上，停了三秒。

理智说：别开，惹麻烦。
良心说：开吧，不然她可能会死。

他深吸一口气，拉开了门。
```

**技巧**：
- 两个选择
- 各有利弊
- 用"理智说/良心说"展示内心声音
- 最后做出选择

---

## 类型2：情感冲突

```
他应该恨她。

三年前，她骗了他，害他差点死了。他找了她三年，想过无数次重逢的场景——他要质问她，要她解释，要她道歉。

但现在她就在他面前，他却问不出口。

因为她看起来太瘦了。瘦得颧骨突出，眼窝深陷，像是这三年也没好过。因为她手上的疤还在，那道替他挡刀留下的疤。

他该恨她的。但看到她发抖的手，他发现自己更想知道——这三年，她是怎么过来的？
```

**技巧**：
- 应该vs实际
- 理性（恨她）vs感性（心疼）
- 用细节触发情感（瘦、疤、发抖）

---

## 类型3：身份冲突

```
陈默看着那张纸条，手在抖。

如果纸条上写的是真的，那他就是那个组织的成员——那个他追查了三年的、害死他师父的组织的成员。

他想否认。他想告诉自己，这是假的，是陷阱。

但他想起师父死前看他的眼神。那种——不是愤怒，是悲伤的眼神。像是早就知道了什么。

难道师父也知道？

如果这是真的，那他这三年的追查算什么？复仇？还是清理门户？

他到底是谁？
```

**技巧**：
- 身份颠覆
- 过去vs现在
- 自我怀疑（"他到底是谁"）

---

## 技巧总结

### 1. 两难选择
- A选项：好处+坏处
- B选项：好处+坏处
- 让读者理解为什么难选

### 2. 内心声音
```
理智说：...
情感说：...
```
或
```
他应该...，但他...
```

### 3. 延迟决定
- 不要立即做出选择
- 展示挣扎过程
- 最后的选择才有分量

### 4. 用细节触发
- 不是无缘无故的冲突
- 用一个细节触发（一封信、一个眼神）

---

## 避免错误

❌ **不要**：
- "他很矛盾"
- "他很纠结"
- "不知道该怎么办"

✅ **要**：
- 展示具体的两难
- 展示内心声音
- 用动作展示（"手在抖"、"停了三秒"）
