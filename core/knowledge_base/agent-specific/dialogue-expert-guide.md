# 对话专家指南（DialogueExpert 方法论）

## 语言特征六维度

### 1. 词汇偏好
- 角色喜欢用什么类型的词（文雅/粗俗/专业/口语化）
- 角色的口头禅/习惯用语
- 角色是否会用行业术语/方言

### 2. 句式结构
- 长句还是短句
- 完整句还是省略句
- 陈述句多还是疑问句多
- 主动句多还是被动句多

### 3. 语气词
- 角色常说什么语气词（啊/哦/嘛/吧/哼）
- 不同情绪下的语气词变化
- 语气词的使用频率

### 4. 说话速度
- 快语速：短句、连珠炮式
- 慢语速：长句、停顿多、思考多
- 不同情绪下的语速变化

### 5. 表达习惯
- 直接表达还是委婉表达
- 喜欢比喻还是直说
- 喜欢反问还是陈述
- 是否有说话时的小动作

### 6. 知识范围
- 角色能说什么不能说什么（信息边界）
- 角色的专业领域
- 角色的文化背景限制

## 潜台词分析

### 核心原则
- 言外有意，不直白说教
- 对话要有「水面下的冰山」
- 角色说的话 ≠ 角色真正想说的话

### 常见潜台词技巧
1. **指桑骂槐**：表面说A，实际说B
2. **欲言又止**：想说但没说出来（读者能感觉到）
3. **反话**：说「没事」但实际有事
4. **沉默**：不说话也是一种表达
5. **转移话题**：回避 = 有隐情

## 对话节奏控制

### 密度
- 对话密集 = 节奏快（适合紧张/冲突场景）
- 对话稀疏 = 节奏慢（适合思考/氛围场景）

### 长度
- 短对话（1-2句）：快节奏、紧张
- 长对话（5句以上）：慢节奏、深入交流
- 交替使用保持节奏变化

## 时代语言审核
- 角色的语言必须符合故事的时代背景
- 古代人不会说「OK」「哈哈」
- 注意避免现代网络用语混入
- 同时代角色的语言风格应该一致

## 每个角色必须有独特的语言指纹
- 不同角色说同样的话应该有不同的方式
- 通过对话能分辨出说话者是谁
- 角色的语言指纹应该与其性格/背景一致
